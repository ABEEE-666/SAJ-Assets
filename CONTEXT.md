# Project Context
SAJ Landing Page

Version : 1.0

---

# Project Overview

Project ini merupakan Landing Page resmi untuk brand **Aqiqah Saung Abah Jajat (SAJ)**.

Landing Page dibuat sebagai media pemasaran utama untuk meningkatkan konversi pengunjung menjadi calon pelanggan melalui WhatsApp.

Project ini bukan:

- Company Profile
- CMS
- Marketplace
- Web Application

Project ini adalah Landing Page yang berfokus pada **Lead Generation**.

---

# Primary Goal

Landing Page harus mampu:

- Membangun kepercayaan.
- Menjelaskan keunggulan layanan.
- Menampilkan paket aqiqah.
- Menampilkan bukti sosial (social proof).
- Menjawab pertanyaan umum.
- Mendorong pengunjung menghubungi WhatsApp.

Seluruh desain dan pengembangan harus mendukung tujuan tersebut.

---

# Business Objective

Target utama Landing Page adalah meningkatkan:

- WhatsApp Click
- Lead
- Konsultasi
- Pemesanan

Setiap perubahan pada Landing Page harus memberikan dampak positif terhadap conversion rate.

---

# Main Conversion

Konversi utama Landing Page adalah:

**WhatsApp Conversation**

Semua CTA, tombol, dan alur halaman harus mengarah pada tujuan tersebut.

Landing Page tidak mengejar:

- Page View
- Like
- Comment
- Share

Keberhasilan utama diukur dari jumlah calon pelanggan yang menghubungi WhatsApp.

---

# Target Audience

Target utama:

- Orang tua muslim.
- Pasangan muda.

- Menginginkan layanan yang praktis.
- Mengutamakan kehalalan.
- Mengutamakan kualitas masakan.
- Menginginkan pelayanan profesional.
