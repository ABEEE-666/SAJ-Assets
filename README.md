# SAJ Landing Page

> AI-Friendly Single HTML Landing Page menggunakan **LP Engine v1.0**

---

# Overview

Repository ini berisi source code Landing Page resmi **Aqiqah Saung Abah Jajat (SAJ)**.

Landing Page dibangun menggunakan **LP Engine v1.0** dengan pendekatan **Single HTML Architecture**, sehingga seluruh website dapat dijalankan dan di-deploy hanya menggunakan satu file HTML.

Project ini dirancang agar:

- Mudah dipelihara manusia.
- Mudah dipahami AI.
- Mudah dipindahkan ke berbagai hosting.
- Tidak memerlukan build process.
- Tidak bergantung pada framework JavaScript.

---

# Quick Start

Sebelum melakukan perubahan pada project, baca dokumentasi berikut secara berurutan.

```
README.md
    ↓
CONTEXT.md
    ↓
docs/engine/LP_ENGINE_v1.md
    ↓
docs/engine/AI_RULES.md
    ↓
docs/engine/PROJECT_STRUCTURE.md
    ↓
docs/project/BRAND_GUIDE.md
    ↓
docs/project/ROADMAP.md
    ↓
docs/project/PROMPT_GUIDE.md
    ↓
docs/project/CHANGELOG.md
```

Setelah seluruh dokumentasi dipahami, lakukan perubahan pada:

```
index.html
```
